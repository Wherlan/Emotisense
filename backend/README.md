# EmotiSense - AI Video Performance Analytics
Created by Wolf of Few Streets

Overview

EmotiSense is an AI-powered platform that analyzes video performances to provide actionable feedback on:
- Facial expressions and micro-emotions
- Voice quality and confidence
- Eye contact and engagement
- Speaking pace and clarity

Built with Python (FastAPI), OpenCV, librosa, and React.

Features

✅ Real-time facial expression analysis (7 emotion classes)
✅ Voice sentiment and prosody analysis
✅ Eye contact tracking
✅ Smile authenticity detection
✅ Performance scoring with personalized recommendations
✅ Beautiful, responsive UI

## Tech Stack

Backend:
- FastAPI (Python)
- OpenCV for computer vision
- librosa for audio analysis
- SQLite database

Frontend:
- React
- TailwindCSS
- Lucide Icons
